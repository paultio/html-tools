## About

Each tool is a self-contained HTML file — no build step, no frameworks, no React. Pure HTML, CSS, and vanilla JavaScript.

## Adding a Tool

1. Create a new `.html` file in the repo root
2. Update the index page and this README with the new tool
3. Push to `main` — GitHub Pages deploys automatically
